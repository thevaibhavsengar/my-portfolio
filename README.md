## ⚡ Getting Started

```bash
# Clone the repository
git clone https://github.com/thevaibhavsengar/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Then open **[http://localhost:3000](http://localhost:3000)** to view it locally.
